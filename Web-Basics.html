Git को समझना (Step-by-Step)

दीप सिंह यादव द्वारा निर्मित नोट्स

1. Working Directory

जहाँ आप कोड लिखते हैं।

git status
→

2. Staging Area

तैयारी का क्षेत्र (Waiting Area)।

git add .
→

3. Local Repo

आपके कंप्यूटर में सेव (Snapshots)।

git commit -m "msg"
→

4. Remote (GitHub)

दुनिया के लिए उपलब्ध (Cloud)।

git push
